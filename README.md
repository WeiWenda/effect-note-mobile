## Installation
```
gem install cocoapods
npm install -g @ionic/cli @capacitor/assets
```

## Debug UI Component
```
npm i -g @ionic/cli
ionic serve
```

## iOS Build
```
npm i
npm run build
npx cap sync ios
npx cap open ios
```

## Android Build
```
npm i
npm run build
npx cap sync android
npx cap open android
```
